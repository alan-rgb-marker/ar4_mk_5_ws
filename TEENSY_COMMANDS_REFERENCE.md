# AR4 Teensy 6.7 韌體命令參考

## 系統命令

### HO - Hello 命令
辨識和連接機器人
```
HO
```

### RB - 重啟
重啟 Teensy 微控制器
```
RB
```

### DB - 偵錯命令
設置偵錯參數
```
DB[D]1[P]1   # 啟用偵錯模式並持久化保存
DB[D]0       # 關閉偵錯模式
```

### SR - 設置機器人 ID
存儲硬體和版本資訊到 EEPROM
```
SR[M]AR4[V]6.7[B]Teensy41[S]ABC123[A]TAG001
```

## 運動命令

### MJ - 關節空間運動
在關節空間中運動到目標位置
```
MJ A-10 B20 C30 D40 E50 F60 J7 0 J8 0 J9 0 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
```

### ML - 直線運動 (Linear)
在笛卡爾空間中執行直線運動
```
ML X100 Y200 Z300 Rz0 Ry0 Rx0 J7 0 J8 0 J9 0 Sp50 Ac10 Dc10 Rm20 Rnd0 W2 Lm000000 Q0
```

### MA - 弧形運動 (Arc)
執行弧形運動（通過中間點）
```
MA Bx150 By200 Bz350 Ex200 Ey250 Ez400 Sp50 Ac10 Dc10 Rm20 RndVal W2 Lm000000
```

### MC - 圓形運動 (Circle)
執行圓形軌跡運動
```
MC Cx150 Cy250 Cz300 Rz0 Ry0 Rx0 Bx100 By200 Bz350 Px200 Py300 Pz250 Sp50 Ac10 Dc10 Rm50 W2 Lm000000
```

### MV - 視覺偏移運動
基於視覺辨識的偏移運動
```
MV X100 Y200 Z300 Rz0 Ry0 Rx0 J7 0 J8 0 J9 0 Sp50 Ac10 Dc10 Rm20 Vr15 W2 Lm000000
```

### RJ - 關節旋轉運動
旋轉特定關節
```
RJ A10 B20 C30 D40 E50 F60 J7 0 J8 0 J9 0 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
```

### MG - G代碼運動
從 G 代碼執行運動
```
MG X100 Y200 Z300 Rz0 Ry0 Rx0 J7 0 J8 0 J9 0 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
```

## Jog（手動控制）命令

### LC - 笛卡爾 Jog (Live Cartesian)
連續笛卡爾空間手動控制，持續到收到停止命令
```
LC V10 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
# V10=x負, V11=x正, V20=y負, V21=y正, V30=z負, V31=z正
# V40=rz負, V41=rz正, V50=ry負, V51=ry正, V60=rx負, V61=rx正
```

### LJ - 關節 Jog (Live Joint)
連續關節空間手動控制
```
LJ V10 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
# V10=J1負, V11=J1正, V20=J2負, V21=J2正, ... V90=J9負, V91=J9正
```

### LT - 工具 Jog (Live Tool)
基於工具座標系的手動控制
```
LT V10 Sp50 Ac10 Dc10 Rm20 W2 Lm000000
```

### JT - 工具增量運動
單次工具座標增量運動
```
JT X0 10 Sp50 Ac10 Dc10 Rm20 Lm000000
# X0=移動座標x, 10=距離, 用於 X0/X1, Y0/Y1, Z0/Z1, R0/R1, P0/P1, W0/W1
```

```

```

## Modbus 條件命令

### WJ - 等待 Modbus 線圈
等待線圈達到特定值（超時）
```
WJ A1 B0x1000 C1 D10
# A=從機, B=線圈, C=期望值, D=超時(秒)
```

### WK - 等待 Modbus 輸入
等待輸入達到特定值（超時）
```
WK A1 B0x1000 C1 D10
```

### SC - 設置 Modbus 線圈
```
SC A1 B0x1000 C1
# A=從機, B=線圈, C=值
```

### SO - 設置 Modbus 輸出暫存器
```
SO A1 B0x1000 C100
# A=從機, B=暫存器, C=值
```

## 運動參數說明

### 速度類型 (SpeedType)
- `Sp` - 速度字首
- `s` - 秒 (指定完成運動的秒數)
- `m` - 毫米/秒 (指定連續速度)
- `p` - 百分比 (0-100%)

### 加速度參數
- `Ac` - 加速比例 (%)
- `Dc` - 減速比例 (%)
- `Rm` - 斜坡參數 (加速度斜率)

### 系統座標
- `X`, `Y`, `Z` - 笛卡爾座標 (mm)
- `Rx`, `Ry`, `Rz` - 旋轉角度 (度)
- `A`, `B`, `C`, `D`, `E`, `F` - 關節角度 (度)
- `J7`, `J8`, `J9` - 外部軸位置

### 腕部配置 (W)
- `W0` - 不指定
- `W1`, `W2`, `W3`, `W4`, `W5`, `W6`, `W7`, `W8` - 不同腕部配置

### 循環模式 (Lm) - 6位二進制
每位表示一個關節是否為連續循環模式：
- `Lm000000` - 所有關節非循環
- `Lm100000` - J1 循環模式

## 端點符號
- 所有命令以 `\n` (換行符) 結束
- 響應通常返回 "Done" 或特定數據值
